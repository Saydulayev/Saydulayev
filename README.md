### **MontWerk e.U. - Веб-сайт**

Профессиональный веб-сайт для компании по монтажу кухонь и мебели в Вене. Полностью разработан с нуля с использованием современных веб-технологий.

**Основные особенности:**
- ✨ Современный адаптивный дизайн с поддержкой темной/светлой темы
- 📱 Полностью адаптивная верстка (Mobile-First подход)
- 🔍 Полная SEO-оптимизация (мета-теги, Open Graph, Twitter Cards, структурированные данные JSON-LD)
- ♿ Высокий уровень доступности (ARIA-атрибуты, семантический HTML, поддержка screen readers)

- 🍪 Cookie баннер с поддержкой GDPR
- 📞 Интеграция с WhatsApp и контактными формами
- 🗺️ Sitemap.xml и robots.txt для поисковых систем

**Технологии:** HTML5, CSS3, Vanilla JavaScript, SEO, Accessibility
